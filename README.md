# Disable Armor Sound

一个简单的Minecraft资源包，用于禁用盔甲穿戴音效。

## 功能

- 完全移除盔甲穿戴、脱下或切换时的默认音效
- 兼容大多数Minecraft版本（1.15 - 26.1.2）
- 不影响游戏其他音效或性能

## 安装方法

1. 下载最新的资源包文件（从[Releases](https://github.com/Junhaoer/disable-armor-sound/releases)页面）
2. 打开Minecraft，进入"选项" > "资源包" > "打开资源包文件夹"
3. 将下载的文件放入文件夹中
4. 在游戏中启用该资源包即可

## 支持的盔甲类型

- 通用盔甲 (generic)
- 锁甲 (chain)
- 钻石甲 (diamond)
- 金甲 (gold)
- 铁甲 (iron)
- 皮革甲 (leather)
- 下界合金甲 (netherite)
- 海龟壳 (turtle)
- 鞘翅 (elytra)
- 铜甲 (copper)

## 注意事项

- 该资源包仅在本地端生效，服务器端的玩家需要各自安装
- 坚守者 (Warden) 仍会听到盔甲穿戴音效
- 不会影响其他游戏音效或性能
